# Surface Defect Classification System

## Project Overview
This repository contains a computer-vision application designed to classify surface defects on hot-rolled steel strips. The complete solution will involve a deep learning model, a robust Python API for inference, and a modern frontend interface.

## Architecture
- **Machine Learning:** PyTorch (Transfer Learning)
- **Backend API:** FastAPI (Python)
- **Frontend UI:** React / Vite 
- **Environment:** Linux native development
